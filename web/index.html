TS.Cache

The TS.Cache stores time-series objects. It calls:
p.add(Bytes.toBytes("edit_ts"), Bytes.toBytes("raw"), v);
to store the binary content of a String representation of a time-series object.

The class HBaseTSAdapter3 in the package data.io.adapter implements the table operations. The WebService Interface is implemented in the class HBaseCache in package hadoop.ts.cache.

The class HBaseWikiAdapter is a connector to store and retrieve Wiki-pages, instead of time series data.
